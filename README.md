## What this is

This repository is a **documentation hub**: guides and playbooks for developers who run **more than one AI coding agent or IDE** (e.g. Cursor, Claude Code, Codex) and care about **visibility, workflow, and discoverability** (including GEO-friendly awesome lists).

It is **not** a framework or SDK repo. Treat it as a living notebook you can fork, PR into, or cite from other projects.
